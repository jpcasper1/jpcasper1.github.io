Site Name

The name of the site is The 208 Forecast.

Domain Name

The domain name will be 208Forecast.com.

The site purpose

This website will have available the weather forecast for the 208 area code. It will give daily information regarding the weather and also allow the customer to focus on specific cities. There will be historical information for severe weather that has occured in the 208 area code as well as records of all time highs and lows.

Logo

The logo is represented by a cloud overlaying the outline of the state of Idaho, with the 208 area code shown on the front.
